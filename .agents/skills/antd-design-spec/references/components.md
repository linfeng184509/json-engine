## 导航组件

### 导航菜单 (Menu)

#### 顶部导航

```
┌─────────────────────────────────────────────────────────┐
│ [Logo]    首页    产品    解决方案    文档    定价       │
└─────────────────────────────────────────────────────────┘
```

**使用场景**：
- 门户性质、前台应用
- 信息层级简单明了

**规范**：
| 项目 | 要求 |
|------|------|
| 类目数量 | 2-7 个 |
| 标题长度 | 4-15 字符 |
| 中文长度 | 2-6 字 |

#### 侧边导航

```
┌──────┬──────────────────────────────────────────────────┐
│ ○    │                                                  │
│ 概览  │                                                  │
│      │                                                  │
│ □    │                                                  │
│ 列表  │              内容区域                            │
│      │                                                  │
│ ◇    │                                                  │
│ 分析  │                                                  │
│      │                                                  │
│ ⚙    │                                                  │
│ 设置  │                                                  │
└──────┴──────────────────────────────────────────────────┘
```

**使用场景**：
- 管理性质应用
- 信息层级多
- 操作切换频率高

**规范**：
- 类目数量不限
- 支持多级展开
- 可配合滚动条

### 面包屑 (Breadcrumb)

```
首页 / 产品管理 / 产品列表 / 产品详情
```

**作用**：
- 告知用户当前位置
- 展示父子级页面关系

**规范**：
| 项目 | 建议 |
|------|------|
| 最大层级 | 5 级 |
| 显示层级 | 3 级以内 |
| 使用原则 | 导航清晰时可不使用 |

### 标签页 (Tabs)

#### 样式选择

| 样式 | 示例 | 使用场景 |
|------|------|---------|
| **基本样式** | ┌─[标签1][标签2]─┐ | 主功能切换，引领整个页面 |
| **卡片样式** | ┌[标签1]┬[标签2]┐ | 页面局部展示，隔离内容 |
| **胶囊样式** | ( 标签1 )( 标签2 ) | 小版块内，与其他组件结合 |
| **竖状样式** | 标签1 │<br>标签2 │ | 分类较多，扩展性好 |

**规范**：
- 标签标题：2-6 个中文字
- 平级内容分类
- 不必跳转页面

### 步骤条 (Steps)

```
横向步骤条:

  ①──────────②──────────③
 完成       进行中      等待

竖向步骤条:

  ① 完成
  │  描述文字
  │
  ② 进行中
  │  描述文字
  │
  ③ 等待
```

**规范**：
| 项目 | 要求 |
|------|------|
| 步骤数 | 2-5 步 |
| 文字长度 | ≤12 字符 |
| 竖向 | 适合多步骤或动态变化 |

**使用场景**：
- 任务复杂
- 存在先后关系
- 需要流程预期

### 分页器 (Pagination)

| 样式 | 使用场景 |
|------|---------|
| **标准样式** | 信息条目多，与表格、卡片搭配 |
| **迷你样式** | 卡片、浮层 |
| **简易样式** | 10 页以内，统计类表格 |

**规范**：
- 超过 5 页提供快速跳转
- 可自定义每页行数
- 页面不足一页不显示

## 数据录入组件

### 文本录入

#### 输入框 (Input)

```
┌─────────────────────────────────────┐
│ 请输入内容                          │
└─────────────────────────────────────┘
```

**使用场景**：较少字符，单行输入

**增强功能**：
- 暗提示（placeholder）
- 前缀/后缀图标
- 清除按钮
- 字数限制

#### 文本域 (Textarea)

```
┌─────────────────────────────────────┐
│                                     │
│  请输入多行内容                      │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

**使用场景**：长篇幅、多行文本

#### 搜索框 (Search)

```
┌─────────────────────────────────────┬───┐
│ 请输入搜索关键词                     │ 🔍 │
└─────────────────────────────────────┴───┘
```

**使用场景**：海量信息中快速定位

### 选择录入

#### 单选框 (Radio)

```
○ 选项A    ○ 选项B    ○ 选项C

或

○ 选项A
  描述文字
○ 选项B
  描述文字
```

**使用场景**：
- 互斥选项
- 选项全部可见
- 需要比较选择

**规范**：
- 选项数量：2-5 个
- 超过 5 个考虑下拉选择

#### 复选框 (Checkbox)

```
☑ 选项A    ☐ 选项B    ☑ 选项C

或

☑ 选项A
  描述文字
☐ 选项B
  描述文字
```

**使用场景**：
- 多项选择
- 状态标记
- 单个状态切换

**规范**：
- 需与提交操作配合
- 单个可表示切换

#### 开关 (Switch)

```
开启 ●━━━━━     或     ━━━━━● 关闭
```

**使用场景**：单个选项状态切换

**规范**：
- 切换立即生效
- 无需操作按钮配合
- 内联标签清晰（启用/禁用）

#### 下拉选择 (Select)

```
┌─────────────────────────────────────┐
│ 请选择                        ▼    │
├─────────────────────────────────────┤
│ 选项 1                              │
│ 选项 2                              │
│ 选项 3                              │
└─────────────────────────────────────┘
```

**使用场景**：
- 选项 >5 项
- 节省空间
- 单选或多选

**规范**：
- 列表按逻辑排序
- 内容显示完整

#### 滑块 (Slider)

```
○━━━━━━━━━━━━━━━━━━━━━━━━━━━━●━━━━━○
0                           50    100
```

**使用场景**：
- 连续区间值选择
- 强度等级设置（音量、亮度）

**规范**：
- 不需精确值：连续滑块
- 需精确值：搭配数字输入框

#### 穿梭框 (Transfer)

```
┌─────────────┐     ┌─────────────┐
│ 选项 A      │     │ 已选 D      │
│ 选项 B      │  >  │ 已选 E      │
│ 选项 C      │  <  │ 已选 F      │
└─────────────┘     └─────────────┘
```

**使用场景**：两栏间移动元素

### 日期选择

#### 日期选择器 (DatePicker)

```
┌─────────────────────────────────────┐
│ 2024-01-15                    📅   │
└─────────────────────────────────────┘
```

**类型**：
- 日期选择
- 日期范围选择
- 日期时间选择

**使用场景**：浏览和选择日期或范围

### 文件上传 (Upload)

| 类型 | 使用场景 |
|------|---------|
| **简单点击上传** | 单文件，不需要预览 |
| **缩略图上传** | 图片文件，需要预览 |
| **拖拽上传** | 大文件，批量上传 |

**规范**：
- 明确文件大小限制
- 明确文件格式
- 明确上传进度

## 数据展示组件

### 表格 (Table)

```
┌─────────┬─────────┬─────────┬─────────┐
│  列1    │  列2    │  列3    │  操作   │
├─────────┼─────────┼─────────┼─────────┤
│  数据1  │  数据2  │  数据3  │ 编辑 删除│
├─────────┼─────────┼─────────┼─────────┤
│  数据4  │  数据5  │  数据6  │ 编辑 删除│
└─────────┴─────────┴─────────┴─────────┘
```

**使用场景**：
- 信息收集和展示
- 数据分析和归纳
- 操作结构化数据

**规范**：
- 时间、状态、操作栏保持完整
- 空数据显示 `-`
- 支持排序、搜索、筛选、分页

**搭配功能**：
- 排序
- 筛选
- 搜索
- 分页
- 选择

### 列表 (List)

```
┌─────────────────────────────────────┐
│ [头像] 标题                          │
│        描述文字                      │
├─────────────────────────────────────┤
│ [头像] 标题                          │
│        描述文字                      │
├─────────────────────────────────────┤
│ [头像] 标题                          │
│        描述文字                      │
└─────────────────────────────────────┘
```

**使用场景**：
- 快速扫读
- 展示空间有限
- 弹窗、侧栏、下拉面板

**特点**：
- 兼顾浏览性与展示性
- 有层次展示内容

### 卡片 (Card)

```
┌─────────────────────────────────────┐
│ [图片]                              │
│                                     │
│ 标题                                │
│ 描述文字                            │
│                                     │
│ ┌─────┐ ┌─────┐                    │
│ │操作1│ │操作2│                    │
│ └─────┘ └─────┘                    │
└─────────────────────────────────────┘
```

**使用场景**：
- 轻量级信息
- 个性化内容
- 信息集中化

**规范**：
- 一行最多 4 个
- 注意信息间距
- 过长信息做截断

### 卡片列表 (Card List)

```
┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│             │ │             │ │             │
│   卡片1     │ │   卡片2     │ │   卡片3     │
│             │ │             │ │             │
└─────────────┘ └─────────────┘ └─────────────┘
```

**使用场景**：
- 强调展示性
- 对象展示机会均等
- 页面吸引力

**特点**：
- 网格布局
- 无特定浏览顺序
- 适合突出对象

### 折叠面板 (Collapse)

```
┌─────────────────────────────────────┐
│ 标题 1                          ▼  │
│ ┌─────────────────────────────────┐ │
│ │ 展开内容                       │ │
│ │ ...                            │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│ 标题 2                          ▶  │
├─────────────────────────────────────┤
│ 标题 3                          ▶  │
└─────────────────────────────────────┘
```

**使用场景**：
- 信息分组收纳
- 导航结构
- 冗长无规则内容

**规范**：
- 关联度低时使用手风琴模式
- 递进式获取信息

### 树形控件 (Tree)

```
├─ 根节点
│  ├─ 子节点1
│  │  ├─ 叶子节点1
│  │  └─ 叶子节点2
│  └─ 子节点2
│     └─ 叶子节点3
└─ 根节点2
```

**使用场景**：
- 层级关系展示
- 文件夹结构
- 组织架构
- 分类导航

### 时间轴 (Timeline)

```
●── 2024-01-15
│   事件描述
│
●── 2024-01-14
│   事件描述
│
●── 2024-01-13
    事件描述
```

**使用场景**：
- 时间流信息
- 事件记录
- 历史追踪

### 走马灯 (Carousel)

```
┌─────────────────────────────────────┐
│                                     │
│           内容 1                    │
│                                     │
├─────────────────────────────────────┤
● ○ ○ ○ ○                            │
└─────────────────────────────────────┘
```

**使用场景**：
- 平级内容并列展示
- 图片或卡片轮播
- 官网首页、产品介绍

**规范**：
- 数量 3-5 个
- 提供数量和方向暗示

## 反馈组件

### 提示信息

#### 警告提示 (Alert)

```
┌─────────────────────────────────────┐
│ ⚠ 警告提示                   [×]   │
│ 这是一条警告信息                      │
└─────────────────────────────────────┘
```

**特点**：
- 非阻断式
- 停留页面位置
- 不自动消失
- 可手动关闭

**类型**：成功、信息、警告、错误

#### 通知提醒 (Notification)

```
┌─────────────────────────────────┐
│ 标题                       [×] │
│ 这是一条通知信息                  │
│                          [操作] │
└─────────────────────────────────┘
```

**特点**：
- 右上角显示
- 全局性通知
- 可承载复杂内容

#### 徽标数 (Badge)

```
┌─────┐
│ 消息 │●5
└─────┘

或

● (红点)
```

**使用场景**：
- 消息提示
- 未读标记
- 状态指示

**规范**：
- 重要信息：数字精确提示
- 权重不高：红点提示

#### 文字提示 (Tooltip)

```
鼠标悬停显示
        ↓
┌─────────┐
│ 提示文字 │
└─────────┘
```

**特点**：
- 简单描述
- 鼠标移入显示
- 不承载复杂内容

#### 气泡卡片 (Popover)

```
        鼠标悬停/点击显示
                ↓
┌─────────────────────┐
│ 标题               │
│ 内容               │
│ [操作1] [操作2]    │
└─────────────────────┘
```

**特点**：
- 承载更复杂内容
- 可包含链接、按钮

### 过程反馈

#### 进度指示

| 组件 | 使用场景 |
|------|---------|
| **Spin** | 加载状态 |
| **Progress** | 进度展示 |

**规范**：
- 加载 >2s 显示
- 可提供取消操作
- 长时间显示当前进度

#### 气泡确认框 (Popconfirm)

```
┌─────────────────────────────────────┐
│ 确定要删除吗？                       │
│               [取消]  [确定]        │
└─────────────────────────────────────┘
```

**特点**：
- 轻量级确认
- 在目标元素附近显示
- 比 Modal 更轻量

### 结果反馈

#### 全局提示 (Message)

```
┌─────────────────────────────────────┐
│ ✓ 操作成功                          │
└─────────────────────────────────────┘
```

**特点**：
- 顶部居中
- 自动消失（默认 3s）
- 轻量级提示
- 不打断操作

**类型**：成功、错误、警告、信息

#### 对话框 (Modal)

```
┌─────────────────────────────────────┐
│ 标题                           [×] │
├─────────────────────────────────────┤
│                                     │
│              内容                   │
│                                     │
├─────────────────────────────────────┤
│               [取消]  [确定]        │
└─────────────────────────────────────┘
```

**特点**：
- 页面居中
- 阻断操作
- 重要反馈

**规范**：
- 传递重要且可操作信息时使用
- 避免滥用

## 按钮设计

### 按钮类型

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   主按钮:    ┌─────────┐                                │
│             │  主要操作 │   ← 一个区域最多一个          │
│             └─────────┘                                │
│                                                         │
│   次按钮:    ┌─────────┐                                │
│             │  次要操作 │   ← 最安全的选择              │
│             └─────────┘                                │
│                                                         │
│   文字按钮:  [ 文字操作 ]   ← 弱化，表格操作列          │
│                                                         │
│   图标按钮:  [ + ]  [ ⚙ ]   ← 节省空间，需 Tooltip     │
│                                                         │
│   虚线按钮:  ┌ ─ ─ ─ ─ ┐   ← 引导添加内容              │
│             │  添加    │                                │
│             └ ─ ─ ─ ─ ┘                                │
│                                                         │
│   危险按钮:  ┌─────────┐                                │
│             │  删除    │   ← 警示风险                  │
│             └─────────┘                                │
│                                                         │
│   幽灵按钮:  ┌─────────┐   ← 深色背景上                │
│             │  透明    │                                │
│             └─────────┘                                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 按钮强调程度

```
高 ◀──────────────────────────────▶ 低

主按钮 > 次按钮 > 文字按钮 > 链接
```

### 按钮位置

| 位置 | 使用场景 |
|------|---------|
| **跟随内容** | 放在用户浏览路径中 |
| **工具栏** | 控制特定内容，靠右放置 |
| **Footer** | 详情页推进操作、复杂表单页 |

### 按钮顺序

```
推荐操作 (左) ───────────────────── 风险操作 (右)

[保存]  [取消]  [删除]

对话习惯:
├── 优先询问可能需要的操作
├── 或希望用户执行的操作
└── 最后向用户提供存在风险的操作

方向性:
├── 具有返回意义 → 放左侧
└── 例如：上一步、返回
```

### 按钮分组

```
方法一：按主次折叠
┌─────────────────────────────────────┐
│ [主操作] [次要] [次要] [更多 ▼]     │
└─────────────────────────────────────┘

方法二：平铺 + 分割
┌─────────────────────────────────────┐
│ [组1-1] [组1-2] │ [组2-1] [组2-2]  │
└─────────────────────────────────────┘
```

### 按钮文案

**规则**：
- 必须使用动词
- 与语境紧密关联
- 用语简练

| 推荐 | 避免 |
|------|------|
| 发布 | 确定 |
| 登录 | 提交 |
| 删除 | 下一步 |
| 保存 | 继续 |

## 表单设计

### 表单布局

#### 单列布局（推荐）

```
┌─────────────────────────────────────┐
│ 姓名                                │
│ ┌─────────────────────────────────┐ │
│ │                                 │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 邮箱                                │
│ ┌─────────────────────────────────┐ │
│ │                                 │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 电话                                │
│ ┌─────────────────────────────────┐ │
│ │                                 │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

**优势**：最高效完成任务的布局方式

#### 弱分组布局

```
┌─────────────────────────────────────┐
│ 姓名            邮箱                │
│ ┌───────────┐   ┌───────────────┐  │
│ │           │   │               │  │
│ └───────────┘   └───────────────┘  │
│                                     │
│ 电话            地址                │
│ ┌───────────┐   ┌───────────────┐  │
│ │           │   │               │  │
│ └───────────┘   └───────────────┘  │
└─────────────────────────────────────┘
```

**使用场景**：空间有限，短宽度相关表单项

#### 卡片分组布局

```
┌─────────────────────────────────────┐
│ 基本信息                             │
│ ┌─────────────────────────────────┐ │
│ │ 姓名  邮箱  电话                │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│ 详细信息                             │
│ ┌─────────────────────────────────┐ │
│ │ 地址  邮编  备注                │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

**使用场景**：内容多（>2屏）且可分类归纳

### 表单项设计

#### 标签 (Label)

**规则**：
- 为普通用户提供易懂的文案
- 为专家用户提供专业术语
- 敏感信息使用暗提示说明原因

#### 帮助提示

| 方式 | 使用场景 |
|------|---------|
| **暗提示** | 短提醒，输入框内 |
| **信息图标** | 说明文案较长 |
| **下方提示** | 短于一句的提醒 |
| **Tooltip** | 详细帮助信息 |

#### 验证反馈

```
┌─────────────────────────────────────┐
│ 邮箱 *                              │
│ ┌─────────────────────────────────┐ │
│ │ abc@                          │ │
│ └─────────────────────────────────┘ │
│ ✓ 格式正确                          │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ 邮箱 *                              │
│ ┌─────────────────────────────────┐ │
│ │ abc                           │ │
│ └─────────────────────────────────┘ │
│ ✗ 请输入有效的邮箱地址              │
└─────────────────────────────────────┘
```

**规则**：
- 反馈紧跟相关区块
- 不自动消失
- 错误说明清晰

### 表单类型选择

| 表单类型 | 字段数 | 特点 |
|---------|--------|------|
| 基础表单 | <7 | 单列，一个区域 |
| 弱分组表单 | 7-15 | 同行组合，分组暗示 |
| 卡片分组表单 | >15 | 多卡片，大标题 |
| 分步表单 | 不限 | 线性流程，步骤引导 |
| 可编辑列表 | 动态 | 动态增减，条目编辑 |
